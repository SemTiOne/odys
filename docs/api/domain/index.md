# `odys.domain`

Public domain models and validation helpers.

This package contains the asset, portfolio, scenario, objective, and validation types that make up the user-facing data model.
Use it when you need to define or inspect the core objects that an energy system is built from.

::: odys.domain

## Subpackages and modules

- `entities`
- `exceptions`
- `objective`
- `scenarios`
- `validation`
